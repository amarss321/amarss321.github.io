Tools I Use

Git

Git is a distributed version control system used to track changes in code during software development.

GitHub

GitHub is a cloud-based platform that hosts Git repositories, allowing collaboration on code projects.

Maven

Maven is a build automation tool used for managing project dependencies and building Java-based applications.

Linux

Linux is an open-source operating system widely used in servers, cloud computing, and DevOps.

SonarQube

SonarQube is a tool for continuous code quality inspection, detecting bugs, vulnerabilities, and code smells.

Jenkins

Jenkins is an automation server that enables continuous integration and delivery (CI/CD) in software development.

Docker

Docker is a containerization platform that allows developers to package and deploy applications in isolated environments.

Kubernetes

Kubernetes (K8s) is an open-source system for automating deployment, scaling, and management of containerized applications.

Terraform

Terraform is an Infrastructure as Code (IaC) tool used to define and provision cloud infrastructure.

AWS Cloud

AWS (Amazon Web Services) is a cloud computing platform that provides scalable infrastructure and services.
